Business Problem

The challenge for this data analyst project is outlined below. This has been used continuously to ensure that the right data has been selected, transformed, used in the data visualization which is meant to be passed on to the business users.
"As a Data analyst working at a bike company, you are asked to visualize data that help readers understand how your company performed this year. Your stake holders interested in finding demographics that shape their customers, which will help improve customer purchases probability.”


Data Cleaning & Transformation

To create the necessary data for analysis and making a dashboard, we need to do some cleaning.
Below are some basic steps in my cleaning process.


Create some working tabs


Before doing anything, I created a working sheet to avoid messing something up and running into some problem. I'll also have a tab for pivot tables and one for the final Dashboard.


Remove duplicates


First, I will check and remove duplicates from my dataset.


Replace features with more informing names


I have two features: Marital Status and Gender. I will replace their values with more informative names.


Edit the currency


Edit the Income column's comma and decimal.


Create better features for visualization


I create a new feature: age brackets, which will be more meaningful when we create visualizations. Specifically, anyone older than 54 was labeled "Old", anyone between 31 to under 54 was labeled "Middle Age" and others were labeled "Adolescent".

Create pivot tables

Pivot tables are how you help build the dashboard.
